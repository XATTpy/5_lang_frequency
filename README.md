# Frequency Analysis of Words

A script that displays the top ten most common words in a text file.

# Quickstart

```bash
$ python3 lang_frequency.py /path/to/file/war_and_peace.txt
['the', 'and', 'to', 'of', 'a', 'he', 'in', 'his', 'that', 'was']
```
